# Math Ninja

Math Ninja is a simple and fun math game built with C++ and Qt. The player answers random math questions before the timer runs out. The game keeps track of score, lives, and time.

## Project Overview

Math Ninja is designed as an interactive educational game for kids. It helps players practice basic math skills such as addition, subtraction, and multiplication in a fun game-style format.

## Features

- Random math questions
- Multiple-choice answer buttons
- Countdown timer
- Score tracking
- Lives system
- Restart button
- Simple and colorful user interface

## How to Play

1. Start the game.
2. Read the math question shown on the screen.
3. Click the correct answer from the four choices.
4. Earn points for correct answers.
5. Lose a life if the answer is wrong or time runs out.
6. The game ends when all lives are gone.

## Built With

- C++
